Get started with Rowan.

Quick start

Up and running in minutes.

# Install
pip install rowan
# Scan a local project
rowan scan ./my-project
# Scan with full output
rowan scan ./my-project --format json --output findings.json
# Run the agentic hunt pipeline
rowan hunt ./my-project --llm gemini
