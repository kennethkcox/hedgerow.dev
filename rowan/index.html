Capabilities

AI-aware static analysis.

General-purpose SAST tools don't understand the AI stack. Rowan was built specifically for it — tracking taint flows through LLM SDKs, agent frameworks, and model loading paths.

AI-native taint tracking

Follows user-controlled data through LangChain, LlamaIndex, and direct LLM SDK calls — identifying paths from model output to dangerous execution.

Supply chain analysis

Scans dependencies for typosquatted AI packages, checks model provenance signals, and generates CycloneDX SBOMs for compliance workflows.

Dangerous sink detection

Finds user-controlled inputs reaching command execution, deserialization, SSRF-prone HTTP calls, file access, and AI orchestration entry points.

SARIF output

Results integrate directly with GitHub Code Scanning, VS Code, JetBrains, and any CI pipeline that accepts the standard SARIF format.

Thicket runtime integration

Static findings feed directly into Thicket's runtime engine — code paths Rowan flags automatically receive elevated monitoring at runtime.

Multi-language support

Python-first, with JavaScript support. Covers FastAPI, Flask, Django, Gradio, Streamlit, and the major AI framework surface areas.
