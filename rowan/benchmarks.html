SGLang — April 2026

Three critical RCEs. All missed by existing tooling.

CVSS 10.0 SSTI via chat_template_kwargs

User-controlled input reaches a Jinja2 Template().render() call through apply_chat_template(). Unauthenticated RCE.

CVSS 8.1 dill.loads deserialization in CustomLogitProcessor

User-supplied logit processor deserialized without validation. Arbitrary code execution via crafted payload.

CVSS 10.0 ZMQ serialization regression — incomplete CVE-2026-3060 patch

CVE-2026-3060 was partially patched. Rowan detected the remaining code path. Unauthenticated RCE restored after the fix.

Letta — May 2026

Two critical RCEs in the core tool executor.

Critical Dynamic code evaluation on LLM-controlled strings

CLI interface passes LLM output to a code evaluator. Any LLM response can execute arbitrary Python.

Critical Core tool executor with full built-in access

LLM-generated tool implementations can read the filesystem, spawn processes, and make network calls.
